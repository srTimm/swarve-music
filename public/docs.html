Getting Started

Welcome to the Swarve Music documentation. This guide will help you set up and use the most complete music bot for Discord.

Note: Swarve Music requires voice channel connection and message sending permissions to function properly.

Prerequisites

  • A Discord server where you have administrator permissions
  • A voice channel where the bot can connect
  • A text channel to send commands

Installation

Adding Swarve Music to your server is very simple. Follow these steps:

1

Click "Add to Discord"

Use the invite button on our main page or at the top of this documentation.

2

Select your server

Choose the server where you want to add Swarve Music from the dropdown menu.

3

Authorize permissions

Review and accept the necessary permissions for the bot to function properly.

4

Done!

Swarve Music is now on your server. Use /play to start playing music.

Quick Start

Once Swarve Music is on your server, you can start using it immediately:

Basic Commands 
/play Blinding Lights     # Play a song
/stop                      # Stop and disconnect the bot

Playing Music

Swarve Music supports multiple music sources. You can play from:

YouTube

Search by name or paste a direct YouTube or YouTube Music link.

/play https://youtube.com/watch?v=...

Permissions

Swarve Music needs the following permissions to function properly:

  • Connect: To join voice channels
  • Speak: To play audio
  • Send Messages: To respond to commands
  • Embed Links: To show embeds with information
  • Add Reactions: For interactive controls
  • Use Application Commands: For slash commands
