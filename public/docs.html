What is Swarve?

Swarve is a feature-rich Discord bot designed to help you manage and enhance your Discord server. Whether you need powerful moderation tools, an engaging leveling system, or fun commands to keep your community entertained, Swarve has you covered.

Note

Swarve uses slash commands by default. You can still use traditional prefix commands by setting a custom prefix.

Key Features

  • Moderation - Ban, kick, mute, warn, and manage your server with ease
  • Leveling System - Engage your community with XP and level rewards
  • Music Player - High-quality music playback with queue management
  • Auto Moderation - Automatically filter spam, links, and inappropriate content
  • Welcome Messages - Greet new members with customizable welcome messages
  • Custom Commands - Create your own commands for your server
  • Economy System - Manage coins, balances, and rewards in your server

Quick Start Guide

Follow these simple steps to get Swarve up and running on your Discord server in just a few minutes.

  1. Invite Swarve to your server

    Click the "Add to Server" button and select the server you want to add Swarve to. Make sure you have the Manage Server permission.

  2. Grant the necessary permissions

    Swarve requires certain permissions to function properly. We recommend granting Administrator permissions for full functionality.

  3. Configure basic settings

    Use the /setup command to run the interactive setup wizard, or configure settings manually with /config.

  4. Start using commands

    You're all set! Use /help to see all available commands or explore the Commands page.

Installation

Adding Swarve to your server is straightforward. Follow the steps below to invite the bot.

Required Permissions

For full functionality, Swarve needs the following permissions:

Permission Required For
Manage Messages Moderation commands, auto-mod
Kick Members Kick command
Ban Members Ban and unban commands
Manage Roles Mute, leveling rewards
Manage Channels Lockdown, slowmode
Connect & Speak Music playback
Send Messages All bot responses
Embed Links Rich embed messages

Warning

Make sure Swarve's role is positioned above the roles you want it to manage. Discord's role hierarchy affects what actions the bot can perform.

Bot Settings

Customize Swarve's behavior using the configuration commands. All settings are server-specific.

Configuration Command

Use the /config command to access all bot settings:

Commands 
# View current configuration
/config view

Permission System

Swarve uses a flexible permission system that allows you to control who can use specific commands.

Permission Levels

Level Name Description
0 User Basic commands available to everyone
1 Moderator Moderation commands (kick, mute, warn)
2 Administrator Server management commands
3 Server Owner Full access to all commands

Moderation

Swarve provides a comprehensive set of moderation tools to help you manage your server effectively.

Quick Reference

Command Description Permission
/ban Ban a user from the server Ban Members
/kick Kick a user from the server Kick Members
/mute Timeout a user Moderate Members
/warn Issue a warning to a user Moderator
/purge Delete multiple messages Manage Messages
/lock Lock a channel Manage Channels

Example: Banning a User

Commands 
# Ban with reason
/ban user:@BadUser reason:"Spamming in chat"

# Temporary ban (24 hours)
/tempban user:@BadUser duration:24h reason:"Cooling off period"

Auto Moderation

Automatically protect your server from spam, inappropriate content, and unwanted links.

New Feature

Auto Moderation is a new feature in Swarve v2.0. Some options may change in future updates.

Auto-Mod Features

Commands 
# Add a word to the blacklist
/automod_blacklist_add insult

# Remove a word from the blacklist
/automod_blacklist_remove insult

# Show the server blacklist
/automod_blacklist_list

Frequently Asked Questions

Why isn't Swarve responding to commands?

Make sure Swarve has the necessary permissions and can see the channel you're using. Also verify that your role is above Swarve's role in the hierarchy if using moderation commands.

Can I use Swarve on multiple servers?

Yes! Swarve can be added to as many servers as you want. Each server has its own independent configuration.

Is there a premium version?

Swarve Premium unlocks additional features like custom welcome images, more auto-mod options, and priority support. Visit our website to learn more.
